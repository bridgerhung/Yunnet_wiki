### 響應式變數 `ref`

<!-- - 提供`<script>`裡的資料給模板使用 -->

- 同步一個元件 (component) 中的資料
- 需要先 `import`
- 照常宣告變數，將值用`ref()`包起來
- 在`<script>`中使用需要加上`.value`

```html
<script setup>
import { ref } from "vue"
const reactiveNumber = ref(69)
reactiveNumber.value += 1
</script>

<template>
    <p>Number is {{ reactiveNumber }}</p>
</template>
```

Note:
在模板 (template, HTML) 中使用 JS：

- 在文件中插入值使用雙大括號
- 在任何 Vue 指令（以 `v-` 開頭的標籤屬性）中

### 在模板中響應式變數

- `v-bind` 使 `id` 屬性和 `dynamicId` 資料同步
- `v-model` 同步輸入欄位內容和資料

```html=
<div id="1"></div>
<div v-bind:id="dynamicId"></div>
<input v-model="userInput"></input>
```

Note:
1: id 是常數
2: id 隨著 dynamicId 變動

## HTML

- HyperText Markup Language，超文本標記語言
- 用 `<標籤>` 標記網頁內容

![](https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML/Getting_started/grumpy-cat-small.png)

Note:
HTML（HyperText Markup Language，超文本標記語言）是打造網頁的基石。它表述並定義網頁的內容。「超文本」（HyperText）是指從某個網頁連到其他網頁的連結

HTML 文件中的元素和其他內容文字不同的地方，在於元素名稱本身用「<」與「>」包圍，稱作「標籤」。HTML 標籤不分英文大小寫。也就是說，它們可以寫成英文全大寫、全小寫、或是混在一起。像是 `<title>` 能寫成 `<Title>`,`<TiTlE>`……等等。

---

### 標題和段落

- `<h1>` 到 `<h6>`
  - `<h1>` 代表主標題
  - `<h2>` 代表副標題
  - `<h3>` 代表更次級的副標題, 依此類推
- `<p>` 代表一個段落 (paragraph)

![](https://after-school-assets.s3.amazonaws.com/h1-h6.png)

---

### 粗體與斜體

- `<em>` (emphasis) 會使內容變成*斜體*
- `<strong>` 會使內容變成**粗體**

---

### 區塊與行內

- 區塊
  - 預設以換行表現
  - 伸展到完整寬度
  - 如 `<h1>`、`<p>`
  - 通用標籤是 `<div>`
- 行內
  - 預設不會換行
  - 伸展到完整寬度
  - 如 `<em>`、`<strong>`
  - 通用標籤是 `<span>`

Note:
通用標籤是沒有語意，結構、樣式用途的元素

### 函式

一個函式的定義由一系列的函式關鍵詞組成, 依次為：

- 函式的名稱。
- `()`中，並由逗號區隔的一個函式參數列表。
- `{}`中，用於定義函式功能的一些語句。

```js
function square(number) {
  return number * number;
}
```

---

### 陣列

- 以`[]`表示
- 用於儲存一系列的資料

```js
let animals = ['cat', 'dog', 'fish'];
for (const animal in animals) {
  alert(animal);
}

// OR
for (let i = 0; i < animals.length; i++) {
  alert(animals[i]);
}

// cat
// dog
// fish
```

---

### 物件

- 以`{}`表示

```js
var person = {
  name: ['Bob', 'Smith'],
  age: 32,
  gender: 'male',
  interests: ['music', 'skiing'],
  bio: function () {
    alert(
      this.name[0] +
        ' ' +
        this.name[1] +
        ' is ' +
        this.age +
        ' years old. He likes ' +
        this.interests[0] +
        ' and ' +
        this.interests[1] +
        '.'
    );
  },
  greeting: function () {
    alert("Hi! I'm " + this.name[0] + '.');
  },
};
```
