### 寫在前面

#### 還記得上次設定的 Super User 密碼嗎?

```bash=
su #進入超級使用者
adduser <你的使用者名稱> sudo

passwd # 重設當前使用者的密碼
```

這樣就取得 sudo 權限啦~
如果遇到指令執行失敗，加個 sudo 就能解決

### 快捷鍵

- `Tab` - 自動補完指令或路徑
- `Ctrl + C` - 中斷目前的操作(中斷前景程序)
- `Ctrl + L` - 清理畫面
  - `clear` - 完全清除
- `Ctrl + Q` - 清除目前輸入
- `Ctrl + D` - 離開(等同輸入 exit+enter)
- `Ctrl + Z` - 將目前程序切換到背景
  - 延伸閱讀: [Linux 切換 process 至背景或前景作業 – Ctrl + z](https://bit.ly/3VvF4Sy)

### 隆重介紹 Standard Stream！

讓我們把`echo`的輸出導向至檔案！

```bash=
>  寫入(rewrite)
>> 追加寫入(append)
# 箭頭可反向操作 <, <<

echo hello world！ > 123.txt
cat 123.txt
```

任何有輸出的可以試試看喔~

- 延伸閱讀: [Linux Fundamentals - I/O, Standard Streams, and Redirection.](https://www.putorius.net/linux-io-file-descriptors-and-redirection.html)

### 搜尋神器！ grep

還記得用`cat`顯示過`/etc/passwd`的內容嗎?
我只想查看 root 的資訊怎麼辦?

```bash=
cat /etc/passwd | grep root
```

grep 的比對是用 Regular Expression(正規表達式)
有興趣的可以去看看詳細規則~

- 延伸閱讀: [正規表達式](https://bit.ly/2RfSNuC)
- 延伸閱讀: [Linux 匹配文字 grep 指令用法](https://blog.gtwang.org/linux/linux-grep-command-tutorial-examples/)

### SSH?

##### Secure Shell Protocol

一種安全的傳輸協定
常用於遠端連線

```bash=
ssh <帳號@IP> -p <端口號(預設22)>

馬上來試試看吧
ssh LetMeIn@Anong.moe
密碼 zxcvbasdfgqwert12345
```

- 延伸閱讀: [學習 SSH：選項和配置參數](https://bit.ly/3TuGL0v)

### SCP?

##### 不是控制，收容，保護的組織喔

##### 全名 Secure Copy

基於 SSH 進行 CP(複製檔案)

```bash=
scp <帳號@IP>:<檔案位置> <目標帳號@IP>:<目錄位置>

# 如果是從本機複製至遠端電腦可省略本機帳號、IP與冒號
scp <檔案位置> <遠端帳號@IP>:<遠端目錄位置>
scp <遠端帳號@IP>:<遠端檔案位置> <本機目錄位置>
```
