## Git 的基礎使用

### git clone 從 github 拉取東西下來

```
git clone
```

### git 初始化

:::note
從來沒用過
所以初始化一下吧!
:::

```
git init
```

### Git 目前的狀態

:::tip
顯示目前 Git 所在分支與工作區狀態
:::

```
git status
```

### Git 顯示從古至今的 commit

:::tip
顯示目前 Git 所有過往的 commit  
(提交)
:::

```
git log --oneline --graph --decorate --all
```

## Git repo 的操作

:::info
針對 repository 的基本操作
:::

### 把檔案加入暫存區

:::info
加進去了才能管理~~
:::

1. 塞進去暫存區

```
git add <file_name>
```

2. 後悔了，從暫存區拿出來

```
git rm --cached <file>
```
