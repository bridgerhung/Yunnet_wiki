### CD 目錄穿越
```code= bash

cd <directory> //進到<directory>目錄

cd 'C://program file/java'
```

### list出當前目錄
```code= bash
ls //顯示目錄
ls -a//顯示隱藏文件
ls -l//顯示文件詳細資料
```

### 新增檔案
Don,t touch me
```code = bash
touch abc.py
```

----

### 讀取檔案內容
貓咪
```code=bash
cat abc.py
```

----

### remove 各種東西
ReMove 阿要注意你刪的是啥ㄟ
```code=bash
rm abc.txt
rm -d //刪除資料夾
rm -rf //刪除目錄資料(不要沒事拿來用 尤其是跟目錄 你會死很慘)
```


----

### 常見的目錄代號
1. ~   home directory
2. .   Current directory
3.  ..(兩個)  Upper directory
4. /   root directory

> 試試看這些路徑吧
