# 資訊盒壓接

![資訊盒壓接](./資訊盒壓接.png)

1. 將資訊盒用一字起子拆開
2. 把網路線穿過盒子，將之前的線剪掉，並用剝線器剝皮
3. 把線卡進去
4. 記得把多的線剪掉
5. 蓋起來，盒型記得黏泡綿膠
