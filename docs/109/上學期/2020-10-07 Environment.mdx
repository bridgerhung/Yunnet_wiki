### Linux's History

- 1991: 0.01 首次發布
- 1992: 以 GPL 授權的 Linux 重新發布
- 1994: Linux 1.0 發布
- 1996: Linux 2.0 發布
- 2011: Linux 3.0 發布
- 2015: Linux 4.0 發布
- 2019: Linux 5.0 發布
