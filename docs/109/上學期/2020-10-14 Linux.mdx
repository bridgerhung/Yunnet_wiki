### 基本操作的部分

```shell=
ls
#列出該資料夾的內容

cd <PATH>
#切換資料夾

pwd
#印出目前路徑
```

> 請善用 Tab 的自動補完功能

### 還是基本操作

```shell=
mv <src> <dst>
#移動檔案

cp <src> <dst>
#複製檔案
```

> 這裡更適合用 Tab 補完目錄

### 新建與刪除很重要

```shell=
mkdir <dir>
#新建資料夾，可以一次輸入多個

rm <file>
#刪除檔案

rm -r <dir>
#刪除資料夾，不加參數會掛掉
```

> 用不到的東西砍掉很正常吧~

### stdout 是啥?可以吃嗎?

```shell=
cat <file>
#印出檔案內容

echo "String"
#印出你打的東西
```

> 廢話，當然不能吃，但很重要

### 背景程式好多喔，真礙眼


### 萬般皆下品，唯有權限高

```shell=
su <User>
#切換使用者，要該使用者的密碼

sudo
#取得管理員權限，需要自己的密碼(且帳號在sudo群組)
```

> 沒 sudo 真滴難辦事...

### 是大於小於欸，跟`+<><`有關嗎?

<CodeBlock>
(src) > (dst)
#將src放入dst中

(src) >> (dst) #將 src 加入 dst 中

(src) < (dst) #將 dst 送入 src 中

(src) << (dst) #將 dst 加入 src 中

</CodeBlock>
