### 變數時間

在程式語言中，沒有定義變數，很難做事吧

> 我該怎麼做呢?

---

### 宣告變數的方法

```python=
a = 1
b = True
c = 0.999
d = 'Hello'
```

> 上面的型態依序是:
> int, bool, float, str

---

### 將讀進來的資料相加

```python=
a = int(input("Num 1:"))
b = int(input("Num 2:"))
print(a + b)
```

> 是不是很簡單呢?

---

### 有輸入/輸入，該來點條件了

```python=
if True:
    print("I'm here!!")
```

> 在 Python 中，縮排重要的跟鬼一樣
> 而且也沒有大括號
